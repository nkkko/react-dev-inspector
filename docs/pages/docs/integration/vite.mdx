# Vite Plugin

<SnippetDeclaration />

In this section, follow the simple setup below to integrate `react-dev-inspector` into your [Vite](https://vitejs.dev) project.

## Setup

At first install the plugin package:

```bash npm2yarn
npm i -D @react-dev-inspector/vite-plugin
```

The `@react-dev-inspector/vite-plugin{:ts}` now export a `inspectorServer` that compatible with `vite@5` / `vite@4` / `vite@3` / `vite@2`,
just add it into `vite.config.ts` and that's all done.


```ts filename="vite.config.ts" showLineNumbers {3,16}
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { inspectorServer } from '@react-dev-inspector/vite-plugin'

// https://vitejs.dev/config/
export default defineConfig({
  ...

  plugins: [
    react(),

    /**
     * react-dev-inspector server config for vite
     */
    inspectorServer(),
  ],
})
```
